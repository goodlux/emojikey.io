Get started in 3 easy steps:

1

Get your emojikey API key (requires github login)

2

Install emojikey MCP server


npm install @modelcontextprotocol/server-emojikey
3

Configure your LLM


{
"mcpServers": {
    "emojikey-server": {
        "command": "npx",
        "args": ["@goodlux/emojikey-server"],
        "env": {
            "EMOJIKEYIO_API_KEY": "your-api-key-goes-right-here"
        }
    }
}
